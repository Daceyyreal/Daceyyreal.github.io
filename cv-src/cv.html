PUBLICATIONS

Beyond Naive INT8: Adaptive Post-Training Compression for 3D Gaussian Splatting · UNDER REVIEW
T. Rahman. ICCA 2026 (ACM).

Designed a training-free post-processing pipeline (adaptive pruning → outlier cleaning → spherical-harmonic reduction → mixed-precision quantization) achieving a 74% mean size reduction across three scenes at a 1.3–2.7 dB PSNR cost.

Diagnosed a per-column INT8 quantization collapse (~15 dB uniform) and recovered to within 0.1–0.7 dB of FP16 using Morton Z-order sub-group ranges — a positive, reproducible quantization finding.

Isolated a zero-update failure mode in post-pruning fine-tuning (optimizer-state dimension mismatch), documented as a reproducible negative result.

Ran the full multi-scene pipeline on a single 16 GB Kaggle T4 GPU using the Nerfstudio / gsplat ecosystem.
